Используемые технологиии:
Java, Spring Boot, Thymeleaf, HTML, JPA, Validation, Stream.

Запуск приложения: в поискокой строке ввести localhost:8080

Приложение демонстрирует работу с потоками ввода ввывода(FileReader, FileWriter)
и работу с базой данных через JPA.

При запуске по URL "/" открывается меню приложения
1 Запись нового текста в файл
    Запись производиться в файл расположенный resources/RecordFile/Hello_My_Text.txt
    При отправке формы файл перезаписываеться. Можно отправить пустую форму(файл будет пустым).
2 Вывод текста из файла на HTML страницу
    Вывод информации идет из того же файла. Если он пустой выводится сообщение "Check your file!"
3 Запись нового текста в DB.
    Работает только с английской раскладкой. Проверяеться на наличее min/max кол-ва символов поля Заголовок и Текст.
4 Вывод всех текстов из DB на HTML страницу
    Вывод всех текстов из DB. Возможно удаление любой записи.

DB - код для создания и заполнения DB в папке resources/initDB.sql
